# LIBET - Unity  
## Français  
Le personnage que l'on joue nommé **Libet** est atteint d'une maladie qui modifie ses sens, métaphore de la maladie d'Alzheimer.  
Cette maladie fait que le personnage, peu à peu, n'est plus capable de reconnaître son entourage, ses environnements.  
Progressivement Libet va voir se déformer et se transformer les éléments du jeu, qui vont devenir peu à peu méconnaissables.  

Moteur utilisé: *Unity 6*  

## English  
The character the player controls, named Libet, suffers from an illness that alters their senses, acting as a metaphor for Alzheimer’s disease.  
This illness gradually prevents the character from recognizing the people around them and their environment.  
As the game progresses, Libet will see the game's elements distort and transform, becoming increasingly unrecognizable.  

Engine used: Unity 6
